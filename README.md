# Options Pricing Analytics

This repository contains a quantitative finance project focused on pricing European options using the Black–Scholes model.

## Objective
To understand how option prices are derived and how risk factors such as volatility, time, and interest rates affect trading decisions.

## Key Concepts
- Call and Put Options
- Black–Scholes Model
- Volatility and Risk-Free Rate
- Greeks (Delta, Gamma, Theta, Vega, Rho)

## Tech Stack
- Python
- NumPy
- SciPy
- Streamlit

This project is intended to demonstrate analytical and quantitative skills relevant for trading and research roles.
